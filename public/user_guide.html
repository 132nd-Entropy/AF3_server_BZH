Overview

Welcome to the AlphaFold 3 Job Management Frontend by AG Brügger! This interface allows you to manage jobs, monitor logs, and review completed tasks efficiently. This guide explains the key functionalities and steps to use the application.

Job Queue

The job queue is divided into three sections:

  • Currently Processing: Displays the name of the job currently being processed.
  • Pending Jobs: Lists jobs in the queue waiting for execution, along with their positions.
  • Completed Jobs: Shows finished jobs with their filenames and completion timestamps.

Adding a Job

Follow these steps to add a job:

  1. Enter a Prediction Name (e.g., protein_analysis_01).
  2. Select a Molecule Type (Protein, RNA, DNA, Ligand, or Ion).
  3. Enter the Molecule Sequence or CCD Code:
    • Protein: Amino acid sequence.
    • RNA/DNA: Nucleotide sequence.
    • Ligand: CCD code and specify the amount.
    • Ion: Select Ion type and specify the number of ions.
  4. Optional: From the Optional Input dropdown, select Add Lipids to Predict Membrane Proteins to enable lipid type and amount selections.
  5. Click Start Prediction to add the job to the queue.

Viewing Logs

Logs for the currently running job are displayed in real-time:

  • The logs show the progress and details of the job execution.
  • If no job is running, the log display will show [No logs available].
  • The log viewer automatically scrolls to the latest entry for convenience.

Troubleshooting

If you encounter issues, try the following steps:

  • No Logs Displayed: Ensure a job is running. Refresh the page if necessary.
  • Queue Not Updating: Wait a few seconds, as the queue refreshes every 5 seconds.
  • Invalid Sequence Error: Verify the input format for the molecule type.
  • Backend Not Responding: Ensure the server is running on the correct port (49200).
